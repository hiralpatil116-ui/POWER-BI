# 🛒 E-Commerce Sales Dashboard – Power BI

## 📌 Overview
This project is an interactive Power BI dashboard created for analyzing e-commerce sales data.  
The dashboard provides insights into customer behavior, product performance, occupation analysis, and state-wise sales trends.

---

## 📊 Dashboard Screenshot

![Dashboard Preview](Dashboard_Screenshot.png)

---

## ✨ Features
- KPI cards for Orders, Amount, Age, and Users
- Product category analysis
- Occupation-wise sales insights
- State-wise sales visualization using maps
- Age group and gender analysis
- Interactive and user-friendly dashboard

---

## 🧰 Tools & Technologies
- Power BI
- Excel / CSV Dataset
- DAX
- Data Visualization

---

## 📈 Key Insights
- Clothing category has the highest orders
- Age group 26–35 contributes the most sales
- IT sector customers are highly active
- Multiple Indian states show strong sales distribution
